# 期權交易條款和條件

當您交易選擇權時，您確認：

1. 您已了解選擇權交易具有高度投機性且包含高度風險。

2. 您確認您必須有經濟能力承擔與選擇權合約交易相關的風險，並且必須能夠承受交易此類產品所產生的任何損失，包括您為多頭看跌期權和多頭看漲期權支付的權利金的全部損失期權頭寸；以及與空頭賣權和空頭買權部位相關的保證金維持要求；和交易成本。

3. 您在出售看跌期權和看漲期權時應遵守長橋證券的保證金維持要求，並且不得超過任何交易所或市場或任何其他有管轄權的監管機構規定的頭寸/行使限額。

4. 在進行第一次期權交易之前，您必須向長橋證券確認，您已閱讀並完全理解最新的期權清算公司（"OCC"）披露文件「期權的特徵和風險」（「OCC 文件」）可在此處取得：https://www.theocc.com/Company-Information/Documents-and-Archives/Options-Disclosure-Document。

5. 您確認選擇權合約是在特定期間內交易的。

6. 波動性、流動性和系統故障可能會使特定市場的執行變得極為困難。

7. 您已了解電子交易服務是一個電子系統，因此可能無法使用。

8. 您已確認，如果帳戶淨值不足以滿足保證金維持要求，長橋證券或清算公司可以自行決定清算您的任何頭寸；如果您的損失等於或超過您帳戶申請中指定的投機資本金額，則終止您的交易特權；清算公司可以自動行使到期時價內 0.01 或更多的看跌期權或看漲期權，但沒有義務這樣做。

9. 您已閱讀並充分理解長橋平台選擇權合約交易的保證金維持要求。

10. 選擇權合約受 OCC 規則和條例約束；其他監管機構以及合約交易交易所的具體規則。

11. 購買選擇權合約必須在購買時全額付款，這表示選擇權合約不得以保證金購買；您完全已了解買賣買權和賣權的保證金維持要求以及此類交易所涉及的風險。

12. 選擇權合約由 OCC 發行，您不得自行或與他人共同違反 OCC 或合約交易的特定交易所的規則規定的持倉和行權限制。

13. 如果您在選擇權合約到期之前沒有足夠的淨值來滿足購買或出售標的證券規定的初始保證金要求，執行經紀商可以自行決定清算標的證券頭寸，從而導致來自選擇權合約的行使；LongBridgeSecurities、執行經紀人和清算公司均沒有義務代表您購買或出售標的證券。

14. 您已了解 OCC 將行使權分配給清算公司（例如清算公司），並且您確認，在分配時，您將被要求：(i) 如果是股權期權，則交付或接受所需數量的股份標的證券，或 (ii) 在股票指數選擇權的情況下，以現金支付或接收結算價格。您已了解，在 OCC 向清算公司進行初始轉讓之日起的一天或多天后，您可能不會收到 LongBridgeSecurities 或其清算公司的轉讓通知，並且缺乏此類通知會給未承保的資產帶來特殊風險。您確認您已閱讀並瞭解 OCC 文件第 VIII 章和第 X 章中所述的風險。

15. 如果執行多頭看跌合約導致標的產品被賣空，則您在下單時對訂單負責；您確認只能在符合初始保證金和保證金維持要求的保證金帳戶中執行賣空；在執行賣空之前，長橋證券的清算公司必須能夠代表您借入指定產品（在這種情況下僅適用股票），以便將股票交付給買方；如果已代表您借入證券，但貸方公司隨後向清算公司發出召回通知，則長橋證券的清算公司將嘗試代表您重新借入證券。然而，如果證券無法重新借用，您確認長橋證券可能會在不通知您的情況下，透過在公開市場上以當前市場價格購買股票來彌補您的空頭部位。您須承擔長橋證券產生的所有相關費用。如前所述，空頭股票的市場價值被視為您的保證金帳戶中的借方項目。

16. 帳戶將收到並收取 Longbridge 平台上公佈的每日利率；您確認這些費率如有更改，恕不另行通知。這些利率應針對您欠 LongBridgeSecurities 的所有借方餘額收取，並適用於清算公司根據與 LongBridgeSecurities 達成的購買、持有或交易股票和期權合約的安排而向您提供的所有信貸。在計算信用利息時，帳戶中持有的利息信用和股息應被視為自由信用餘額的一部分。利息計算以一年 365 天為基礎，利用結算日餘額。
