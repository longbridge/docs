# LB Café

LB Café（LONGBRIDGE Café）是长桥证券在香港的线下实体空间，为长桥用户提供咖啡、会议室和金融活动等服务。App 内提供进入 LB Cafe 功能区的入口，用于展示用户码、使用咖啡券和预约空间。

![](./images/CgtBbpJHFoSSYexngadjIhVxpFE.png)

LB Café

## 页面入口

「我的」→「LB Café」

## 门店信息

**地址：** 香港九龙尖沙咀弥敦道 26 号

## 空间区域

| 区域 | 说明 |
| --- | --- |
| BF Lounge | 地下层休闲空间 |
| GF Café | 地面层咖啡区 |
| 1F Space | 一楼活动及会议空间 |

## 用户权益

| 权益 | 说明 |
| --- | --- |
| 5 折咖啡 | 长桥开户用户到店消费享受咖啡 5 折优惠 |
| 精酿啤酒 | 店内提供精酿啤酒 |
| 会议室 | 提供可预约的私人会议室 |
| Free Wi-Fi | 免费无线网络 |
| 金融讲座 | 定期举办金融主题讲座和活动 |
| Lifestyle 活动 | 生活方式相关体验活动 |

## App 内功能

### 用户码

展示本账户专属二维码，到店出示后享受折扣权益及会员服务。

### 咖啡券

查看账户内已获得的咖啡优惠券，到店时出示核销使用。
