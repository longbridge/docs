# 防诈骗提示

钓鱼诈骗防范、官方联系方式、双重认证机制及四招防骗建议。

## 警惕钓鱼诈骗

近期市场上出现冒充金融机构名义发送短信、WhatsApp 信息或钓鱼链接的诈骗行为。长桥证券（香港）官方发出的 WhatsApp 信息及手机短信目前均不包含任何外部链接。

如收到任何自称来自长桥证券但附有可疑链接的信息，请勿点击、勿转发、勿回复，建议第一时间与官方客服核实。

## 双重验证（2FA）

长桥证券（香港）已全面启用双重认证（2FA）机制，从登录、交易到账户操作，全方位保护身份资讯与资金安全。

## 防骗四招

1. **认清官方信息来源**：仅信任通过长桥证券 App、官网及认证社交平台发布的资讯

2. **不轻信陌生信息**：收到自称「长桥客服」的陌生来电或信息，请提高警觉

3. **不提供个人资料**：长桥不会通过电话、电邮或信息主动索取账户资料、验证码或密码

4. **定期检查账户安全**：建议定期检视账户安全记录，并养成更换密码的习惯
